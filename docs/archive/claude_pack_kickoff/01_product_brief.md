# 01 — Product Brief

## Working Title
**Squishy Smash**

## Elevator Pitch
Squishy Smash is a fast, ultra-satisfying casual smash game where players tap, drag, slap, crush, burst, and splat squishy objects across a small play area. The game blends stress-relief loops, meme-style absurdity, juicy audiovisual feedback, and short-session mobile gameplay.

## Product Vision
Create an iPhone game that feels instantly understandable, visually chaotic, and highly screen-recordable for social sharing. The game should be simple enough to build quickly but structured in a way that lets new trending content be added fast.

## Design Goals
- Immediate fun in under 3 seconds
- One-handed mobile play
- Strong visual and audio payoff
- Repeatable micro-sessions
- Easy asset swapping for meme-inspired updates
- Low complexity, high replay value
- Great for TikTok / Reels clips

## Audience
### Primary Audience
- Teens and young adults who enjoy TikTok-style humor, fidgets, satisfying games, and overstimulating sensory loops

### Secondary Audience
- Casual players looking for stress-relief / anti-stress / ASMR mobile games
- Players who liked Rage Smash but want something more colorful and meme-aware

## Core Emotional Promise
Players should feel:
- “That looked satisfying.”
- “I want to do that again.”
- “I want to see what object comes next.”
- “I need to screen record that pop.”

## Market Positioning
This should sit between:
- satisfying / ASMR mobile games
- antistress and fidget apps
- weird meme-inspired casual games
- short-burst destruction games

## Key Differentiators
1. **Rage Smash DNA** — familiar destruction energy
2. **Squishy sensory focus** — not hard smashing only; softer, gooier, poppier interactions
3. **Meme-ready content pipeline** — objects can be updated rapidly
4. **Audio-first juice** — popping, squishing, burst layers, splats, goo drips
5. **Clip-friendly moments** — visual chaos and sudden splat reactions designed for social media

## MVP Feature Set
- Tap-to-smash gameplay
- Drag-and-slam gesture
- 10–20 smashable objects in rotation
- Combo scoring
- Particle splat system
- Unlockable object packs
- Basic progression and coin rewards
- 3 environment themes
- Strong sound layering
- Simple menus and shop

## Post-MVP Features
- Daily challenge
- Limited-time meme packs
- Boss squishies / giant objects
- Wall stain persistence for a single round
- Replay / share highlight moments
- Haptics tuning
- Seasonal packs
- Leaderboards

## Tone and Style
- playful
- absurd
- colorful
- chaotic in short bursts
- satisfying rather than violent
- slightly gross in a funny way, not disturbing

## Important Constraint
Do **not** rely on direct copies of trending IP. Build original objects that are clearly inspired by internet trends but safely distinct.
