# Squishy Smash — Claude Code Branding, Logo, and App Icon Brief

## Purpose
This document tells Claude Code how to use the new visual branding direction for Squishy Smash and how to integrate it cleanly into the product and marketing pipeline.

We now have strong concept direction for:
- a soft-gloss playful logo
- mascot-driven app icons
- a premium cute mobile-game identity

Main branding goal:
Create a visual brand that feels:
- cute
- satisfying
- premium
- collectible
- modern
- highly readable on the App Store
- strong in TikTok/Reels/Shorts thumbnails

---

# Brand personality

Squishy Smash should feel like:
- a toy you want to touch
- a satisfying world you want to stay in
- a playful collectible brand
- a premium casual mobile game
- a modern pastel iOS-native experience

This is **not** supposed to feel:
- gritty
- edgy
- noisy
- childish in a cheap way
- low-effort
- random meme clutter

It should feel:
- soft
- glossy
- rounded
- polished
- comforting
- slightly addictive
- highly visual

---

# Logo direction

## Primary logo concept
Use a rounded bubbly “Squishy Smash” wordmark with:
- soft glossy bubble letters
- pink/cream pastel palette
- subtle splat accents
- playful soft-outline plaque/backing shape
- optional cute mascot peeking above the logo
- premium toy-like 3D rendering feel

## Visual goals
- instantly readable
- memorable
- app-store friendly
- splash-screen friendly
- usable in marketing creatives
- strong on light or gradient backgrounds

## Typography style
The lettering should feel:
- inflated
- soft
- rounded
- slightly squished
- premium and polished

Avoid:
- sharp edges
- thin lines
- overly detailed textures
- complex outlines that break at small size

---

# App icon direction

## Strategic principle
The icon should be simple and character-led.

Do **not** put small text in the app icon.

Use:
- one main mascot
- bold clean background
- bright readable contrast
- premium soft-gloss shading
- strong face/emotion

---

## Current strongest icon routes

### Option A — Bunny/Cream mascot icon
Strengths:
- cute
- friendly
- simple
- highly readable
- broad casual appeal
- feels collectible

Use case:
Best all-around icon if you want a clean, universal casual-game identity.

### Option B — Pink creature icon
Strengths:
- more energetic
- playful
- stronger personality
- more eye-catching
- feels slightly more toy/viral

Use case:
Best if you want the icon to feel more scroll-stopping and expressive.

### Option C — Dumpling reveal icon
Strengths:
- directly aligns with squishy-food trend direction
- strong reveal/payoff feel
- cute and premium
- emotionally warm

Use case:
Best if you want to lean more into the dumpling/squishy reveal theme.

---

# Recommendation hierarchy

## Recommended primary icon
**Test first: Bunny/Cream mascot icon**

Why:
- safest broad-market appeal
- strongest readability
- cutest neutral mascot energy
- easiest to brand across packs and marketing

## Recommended secondary A/B icon
**Pink creature icon**

Why:
- may outperform in ad creatives and social thumbnails
- slightly more expressive and energetic

## Recommended limited/seasonal icon
**Dumpling reveal icon**

Why:
- excellent for themed drops, App Store tests, seasonal updates, and event campaigns

# Visual system guidance

## Core palette
Use the established soft premium palette:
- warm pink
- cream
- pastel blue
- lavender
- soft purple
- occasional gold for rare/reveal moments

## Materials
Everything should feel:
- soft-gloss
- rounded
- toy-like
- smooth
- slightly reflective
- premium and tactile

Avoid:
- realistic dirt/grit
- flat dull materials
- harsh metallics
- noisy patterns

---

# UI alignment
Claude should align the in-game UI to the brand by using:
- rounded corners
- soft progress bars
- subtle glow states
- clean white or cream text
- minimal clutter
- soft shadows
- occasional splat accents
- restrained but premium gradients

The logo and icon should feel like they belong to the exact same world as the game.

---

# App Store considerations

## Icon best practices
The icon should:
- read clearly at small size
- use one central focal point
- have strong contrast
- avoid tiny details
- avoid text
- feel premium immediately

## Logo best practices
The logo should:
- read well on splash and menu screens
- remain legible when scaled down
- have a transparent-background version
- work on both gradient and neutral backgrounds
