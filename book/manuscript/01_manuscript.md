# Squishy Smash: Meet the Squishies — Manuscript v1

**Production target:** 32-page paperback, 8.5 × 8.5 in trim (8.625 × 8.75 in with bleed), full color, full bleed, KDP

**Voice:** playful, warm, magical, easy for children. Lowercase-friendly when it serves rhythm. No sarcasm, no scary edges.

**Word counts:** every page falls in the 30–90 word band recommended by the production brief.

**Art reference:** every character spread maps directly to a file in `assets/cards/final_48/`. File numbers are noted next to each character so the layout pass can drop art with no lookup.

## Open production decisions

These are not blockers for layout; flagging so they don't get forgotten:

1. **Author/publisher line on copyright page** — currently reads "Squishy Smash". Confirm whether to use a personal author name or stay branded.
2. **ISBN** — KDP can assign a free ISBN at publish time, or you can supply one. Decide before final upload.
3. **Title page mascot** — three options proposed (single Soft Dumpling, or 3-mascot cluster of 001+017+033). Pick one.
4. **Cover copy** — back cover blurb + spine treatment is the next manuscript file (`book/cover/cover_copy.md`). Not yet written.
5. **Section dividers** — page-by-page outline does not call for dedicated divider pages; pack intro spreads (9–10, 19–20, 27–28) act as the dividers. If you want stronger separation, we can budget a single divider page per pack and trim one premium card from each section.
