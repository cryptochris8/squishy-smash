# Squishy Smash — Page-by-Page KDP Character Book Outline for Claude

## Book direction
This is the first **Squishy Smash character/world introduction book** for Amazon KDP.

It should be:
- highly visual
- collectible in feel
- easy for young kids to enjoy
- strong enough to begin building the Squishy Smash publishing universe

## Recommended production target
- Format: Paperback
- Interior: Full color
- Interior style: Full bleed
- Trim size: 8.5 x 8.5 inches
- Target length: 32 pages

This outline assumes a **32-page book**, which is a strong sweet spot for KDP and gives enough room for premium visual spreads.

---

# OVERALL BOOK STRUCTURE

## Section flow
1. Front matter
2. Welcome to Squishy Smash
3. Meet the three packs
4. Featured character spreads by pack
5. Closing / favorite-character / teaser pages

# RECOMMENDED CONTENT STYLE FOR CHARACTER ENTRIES

Each character block should be brief.

## Suggested structure per character
- Name
- Pack
- One short descriptive line
- 2 quick fun traits

### Example format
**Soft Dumpling**  
A cozy little squishy who loves gentle bouncing.  
- Warm and kind  
- Puffy and playful

This keeps reading easy for young kids and parents.

# VISUAL HIERARCHY RULES

Claude should prioritize:
1. large character art
2. strong readable name
3. small supporting text
4. clear section labeling
5. lots of breathing room

Do not overcrowd pages with too many tiny card details.

This is a **book**, not a card binder.
